**C# (C Sharp) kya hai?**

C# aik programming zaban hai jo Microsoft ne banai hai. Ye bohat maqbool zaban hai jo mukhtalif qisam ki applications banane ke liye istemaal hoti hai, jaise ke:

*   **Windows applications:** Aap C# se desktop applications bana sakte hain jo Windows operating system par chalti hain.
*   **Web applications:** ASP.NET, jo C# par mabni hai, web applications banane ke liye istemaal hota hai.
*   **Mobile applications:** Xamarin ke zariye, aap C# se iOS aur Android ke liye mobile applications bana sakte hain.
*   **Games:** Unity game engine C# istemaal karta hai, is liye aap C# se games bhi bana sakte hain.
*   **Cloud applications:** Microsoft Azure ke zariye, aap C# se cloud-based applications bana sakte hain.

*   **Asan (Easy):** C# seekhna nisbatan asan hai, khas tor par agar aap ne pehle koi programming zaban seekhi hui hai.
*   **Powerful:** C# bohat powerful zaban hai jo complex applications banane ke liye istemaal ho sakti hai.
*   **Object-oriented:** C# aik object-oriented programming (OOP) zaban hai, jis ka matlab hai ke aap code ko reusable components mein organize kar sakte hain.
*   **.NET framework:** C# .NET framework ka hissa hai, jo bohat sari useful libraries aur tools provide karta hai.
*   **Community:** C# ki aik bari aur active community hai, is liye aap ko help aur support asani se mil sakti hai.

Yeh aik asan C# program hai jo "Hello, World!" print karta hai:

```csharp
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }

```

**Explanation (Wazahat):**

*   `using System;`: Yeh line `System` namespace ko include karti hai, jis mein useful classes aur methods shamil hain.
*   `namespace HelloWorld`: Yeh program ka namespace hai. Namespaces code ko organize karne mein madad karte hain.
*   `class Program`: Yeh program ki class hai. Classes code ko encapsulate karne mein madad karte hain.
*   `static void Main(string[] args)`: Yeh program ka main method hai. Program execution yahin se shuru hota hai.
*   `Console.WriteLine("Hello, World!");`: Yeh line console par "Hello, World!" print karti hai.
